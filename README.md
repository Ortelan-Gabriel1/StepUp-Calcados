# StepUp-Calcados
O projeto StepUp Calçados é uma plataforma digital criada para facilitar a venda de calçados e a organização interna de uma loja

*Para que o projeto funciona?*
O sistema serve para duas frentes principais:
Para o Cliente: Oferece um site de compras prático e intuitivo, onde famílias podem encontrar sapatos, tênis e sandálias com facilidade
O usuário consegue filtrar os produtos por tamanho, cor, gênero e idade (adulto ou infantil), garantindo que encontre o que precisa de forma rápida
Para a Loja: Funciona como uma ferramenta de controle de estoque em tempo real
O sistema ajuda os funcionários a saberem exatamente quantos produtos existem e em qual prateleira ou corredor eles estão guardados

*O que ele faz de principal?*
Busca Facilitada: Permite pesquisar produtos por nome ou categoria através de uma barra de pesquisa dinâmica
Gestão de Estoque: Avisa quando um produto está acabando ou quando está parado há muito tempo no depósito
Segurança e Acesso: Possui áreas restritas com login para diferentes funcionários (vendedores e estoquistas), garantindo que apenas pessoas autorizadas alterem informações importantes
Visual Moderno: Foi planejado para ser elegante e fácil de usar tanto no computador quanto no celular
